# 開発環境の使い方

## ポートの使い分け

このプロジェクトでは、デモ用とバックエンド開発用で異なるポートを使用できます。

### デモ用（ポート4000）
モックアップモードで動作し、DB接続なしでデモ画面を表示します。

```bash
pnpm dev:demo
```

または

```bash
PORT=4000 pnpm dev
```

ブラウザで `http://localhost:4000` にアクセス

### バックエンド開発用（ポート4001）
実際のDB接続でバックエンド開発を行います。

```bash
pnpm dev:backend
```

または

```bash
PORT=4001 pnpm dev
```

ブラウザで `http://localhost:4001` にアクセス

## 同時起動

両方のポートで同時に起動することも可能です：

```bash
# ターミナル1: デモ用（4000）
pnpm dev:demo

# ターミナル2: バックエンド開発用（4001）
pnpm dev:backend
```

## モックアップモードについて

デモ用（4000）では、環境変数 `VITE_OAUTH_PORTAL_URL` と `VITE_APP_ID` が設定されていない場合、自動的にモックアップモードで動作します。

- 認証をスキップ
- ダミーデータを表示
- API呼び出しを無効化

バックエンド開発用（4001）では、実際のDB接続とAPIを使用します。
